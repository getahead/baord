# kanban-board
Simple Task manager with board view for Agile workflow

- Backbone, Marionette 
- Node.js
- MongoDB

### Installation

```sh
$ mongod
$ npm install
$ npm install -g grunt-cli
$ bower install
$ node bin/www
$ grunt build
```
### dev mode
```sh
$ NODE_ENV=development node bin/www
$ grunt
```
